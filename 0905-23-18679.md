Que le da al usuario o usuarios una mejor forma de llevar un control sobre sus proyectos y mayor flexibilidad a la hora de trabajar.

## ¿Qué ventajas crees que tiene usar Git en proyectos de programación?

Usar Git en proyectos de programación ofrece múltiples ventajas clave:



* **Control de versiones:** Permite rastrear cambios en el código, revertir a versiones anteriores y comparar diferencias entre versiones.



* **Colaboración eficiente:** Múltiples desarrolladores pueden trabajar en paralelo en el mismo proyecto sin conflictos, gracias a ramas y fusiones.



* **Historial completo:** Cada cambio está registrado con detalles como quién lo hizo, cuándo y por qué (gracias a los mensajes de commit).



* **Seguridad y backup:** El repositorio actúa como copia de seguridad del código, incluso si el sistema local falla.



* **Flujos de trabajo organizados:** Facilita métodos como GitFlow o GitHub Flow para gestionar desarrollo, pruebas y despliegue.



* **Integración con herramientas:** Se integra con plataformas como GitHub, GitLab y CI/CD para automatizar pruebas y despliegues.



* **Transparencia y auditoría:** Todos los cambios son visibles, lo que mejora la transparencia y facilita la revisión del código.
